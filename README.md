# 🐦 Tweet Sentiment Analysis

This project performs sentiment analysis on tweets using Natural Language Processing (NLP) and Machine Learning techniques. It classifies tweets as **positive** or **negative**.

---

## 🚀 Features

* Text preprocessing (removal of stopwords, stemming)
* TF-IDF vectorization
* Logistic Regression model
* Accuracy evaluation
* Uses real-world dataset (Sentiment140)

---

## 🛠️ Tech Stack

* Python
* Pandas
* NumPy
* NLTK
* Scikit-learn

---

## 📂 Project Structure

```
tweet_sentiment_analysis.py
README.md
.gitignore
```

---

## 📊 Dataset

Dataset used: Sentiment140 from Kaggle

⚠️ Note: You need to download the dataset manually using Kaggle API.

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/tweet-sentiment-analysis.git
```

2. Install dependencies:

```
pip install -r requirements.txt
```

---

## 🔑 Kaggle Setup

1. Go to Kaggle and download your API key
2. Place `kaggle.json` in:

```
~/.kaggle/
```

3. Run:

```
chmod 600 ~/.kaggle/kaggle.json
```

---

## ▶️ Usage

Run the script:

```
python tweet_sentiment_analysis.py
```

---

## 📈 Model

* Algorithm: Logistic Regression
* Vectorization: TF-IDF
* Evaluation Metric: Accuracy Score
